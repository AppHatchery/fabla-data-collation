Fabla Data Toolkit

Aggregate your data and analyze participation patterns with powerful visualization tools.

🔒 Privacy & Security

All data processing happens locally in your browser. No data is sent to external servers, ensuring complete privacy and compliance with research data protection requirements.

📋 Important: File Type Requirements

This tool is designed to work with raw data files that contain individual response records. It will not work properly with 30 day participation summary files. Please ensure you're uploading the raw CSV files containing the actual response data.
